HOW TO DEPLOY THIS WEBSITE FOR FREE:

1. GitHub Pages:
   - Upload the files to a GitHub repository.
   - Go to Settings > Pages and select the branch to deploy.

2. Netlify:
   - Create an account at https://www.netlify.com/.
   - Drag and drop the folder into Netlify's dashboard.

3. Vercel:
   - Create an account at https://vercel.com/.
   - Upload the project using their CLI or drag-and-drop method.
